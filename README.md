# React-Native-App-Demo
## Installation
```
git clone {this repo}
yarn
```
### For ios:
```
cd ios && pod install
```
## Usage
Make sure your device is connected.
### For ios:
```
yarn ios
```
### For android:
```
yarn android
```
## Features
* Language Localization
* Customized Theme
* Gestures
* ...
